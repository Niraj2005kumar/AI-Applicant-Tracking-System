# AI-Powered Applicant Tracking System (ATS)

## 📌 Project Overview

The AI-Powered Applicant Tracking System (ATS) is an intelligent recruitment platform that helps recruiters manage job postings, applications, and candidate screening. The system uses AI to analyze resumes, extract skills, compare them with job descriptions, and rank candidates based on their suitability.

---

## 🚀 Features

- Recruiter Authentication
- Applicant Authentication
- Job Posting & Management
- Public Job Board
- Resume Upload (PDF/DOCX)
- AI Resume Parsing
- AI Candidate Matching
- Candidate Ranking Dashboard
- Application Tracking
- Email Notifications
- Secure Document Storage
